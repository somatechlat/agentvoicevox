# Product Overview

This repository contains two main products:

## 1. OpenVoiceOS Blog (Root)
A statically generated blog built with Next.js showcasing OpenVoiceOS project updates, tutorials, and announcements. Blog posts are written in Markdown with front matter support and stored in `/_posts`.

## 2. AgentVoiceBox Engine (ovos-voice-agent/AgentVoiceBoxEngine)
An enterprise-grade voice agent platform providing OpenAI Realtime API compatibility with open-source infrastructure. Key capabilities:

- **Voice Processing Pipeline**: STT (Faster-Whisper), TTS (Kokoro/phoonnx), LLM integration
- **Multi-tenant SaaS Portal**: Customer dashboard, billing (Lago), usage tracking, API key management
- **Enterprise Features**: Keycloak authentication, OPA policy enforcement, Redis session management
- **Observability**: Prometheus metrics, Grafana dashboards, structured logging

The platform enables organizations to deploy private, cost-effective voice AI infrastructure with full data sovereignty.
