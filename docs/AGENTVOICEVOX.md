# AgentVoiceVox Documentation Index

AgentVoiceVox (aka AgentVoiceBox) is the Django-based voice platform located at `ovos-voice-agent/AgentVoiceBoxEngine`.

Start here:
- Platform overview and features: `ovos-voice-agent/AgentVoiceBoxEngine/README.md`
- Local development guide: `ovos-voice-agent/AgentVoiceBoxEngine/docs/LOCAL_DEVELOPMENT.md`
- REST API docs: `http://localhost:65020/api/v2/docs` (when running)
- WebSocket endpoints: `ovos-voice-agent/AgentVoiceBoxEngine/docs/asyncapi.yaml`

Note: The repo root contains a Next.js blog site and is separate from the platform stack.
